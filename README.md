# Mindo
# Personal Project Management System

Separate **long-term planning** from **daily execution**.

## 🧠 Mind Map
Responsible for project management, goal planning, and structured task breakdown.

- Organize projects
- Define long-term goals
- Break goals into actionable milestones
- Visualize project structure

## 📅 Daily
Focus on what needs to be done today.

- Daily to-do list
- Timeline view
- Calendar synchronization
- Track current tasks and progress

## 🎯 Scenes
Designed for high-frequency routines and repetitive actions.

- Quick Check-in
- One-tap task start
- Built-in Timer
- Habit and routine tracking


把「长期目标拆解」和「每日执行记录」分开的个人项目管理系统。思维导图负责项目、目标、结构化拆解； Daily 负责当天或当前要执行的 Todo、时间线、日历同步； 场景区负责高频行为的快速 Check-in / Timer。
