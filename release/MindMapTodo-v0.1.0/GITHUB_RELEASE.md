## Release description

```text
This is the first downloadable prototype release of MindMap Todo.

Included:
- Mind map based project planning
- Daily Todo
- Check-in and Timer
- Floating timer ball
- Timeline
- Calendar
- Statistics
- Scene management
- Workspace / Folder / Project hierarchy
- Trash restore
- macOS Apple Calendar bridge for Check-in and Timer records
- Reserved native API bridge for Notification, Account, File and Search integrations

Download:
- MindMapTodo-v0.1.0.zip

How to use:
1. Download and unzip.
2. Open app/index.html in a browser.
3. For desktop development, run npm install and npm run start.

Note:
Apple Calendar sync works from the Electron/macOS desktop build after the system permission dialog is approved. Browser-only use cannot write to Apple Calendar. Other native system integrations remain reserved interfaces.
```
